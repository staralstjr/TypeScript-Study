- **Runner**: tsx, ts-node
- **IDE**: VS Code (with JavaScript and TypeScript Nightly)

## 📅 커리큘럼 및 학습 내용

### 📂 Section 1. 타입스크립트 개론
- 타입스크립트 소개 및 개발 환경 구축
- 컴파일러 옵션 설정하기 (`tsconfig.json`)

### 📂 Section 2. 타입스크립트 기본
- **기본 타입**: 원시 타입, 리터럴 타입, 배열과 튜플
- **객체**: 타입 별칭, 인덱스 시그니처, Enum 타입
- **기타**: Any, Unknown, Void, Never

### 📂 Section 3. 타입스크립트 이해하기
- 타입 계층도 (타입은 집합이다)
- 객체 타입의 호환성 및 대수 타입 (Union, Intersection)
- 타입 추론, 단언, 좁히기
- 서로소 유니온 타입

### 📂 Section 4. 함수와 타입
- 함수 타입 표현식과 호출 시그니처
- 함수 타입의 호환성 및 오버로딩
- 사용자 정의 타입 가드

### 📂 Section 5. 인터페이스
- 인터페이스 기본 문법
- 인터페이스 확장(Inheritance) 및 합치기(Declaration Merging)

### 📂 Section 6. 클래스
- JS/TS 클래스 및 접근 제어자 (`public`, `private`, `protected`)
- 인터페이스와 클래스의 활용

### 📂 Section 7. 제네릭 (Generic)
- 제네릭 함수, 타입 변수 응용
- 제네릭 인터페이스 & 타입 별칭 & 클래스
- Promise와 제네릭 활용

### 📂 Section 8. 타입 조작하기
- 인덱스드 엑세스 타입 & keyof 연산자
- 맵드 타입 (Mapped Types)
- 템플릿 리터럴 타입

### 📂 Section 9. 조건부 타입
- 조건부 타입 소개 및 활용
- 분산적인 조건부 타입
- `infer`를 이용한 타입 추론

### 📂 Section 10. 유틸리티 타입
- **Mapped 기반**: Partial, Required, Readonly, Pick, Omit, Record
- **조건부 기반**: Exclude, Extract, ReturnType

### 📂 Section 11. 리액트와 타입스크립트
- React + TypeScript 환경 설정
- State, Props, Context API의 타입 정의
- 외부 라이브러리 사용 및 템플릿 활용
